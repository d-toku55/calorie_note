# アプリケーション名
## Easy Calorie Note

# アプリケーション概要
## このアプリケーションはカロリーを知りたい食品名や料理名を検索し、検索結果をメモできるアプリです。
### 注意: 食品データ、料理名データは合わせて100件しか入っておりません。

# 利用方法
## 未登録のユーザー、未ログインの方はカロリーを知りたい食品名や料理名を検索欄に入力し、検索結果の「食品名や料理名、g、kcal」をご覧いただけます。また新規登録やログインしていただきマイページに移動すると、検索結果の「食品名や料理名、g、kcal」を保存するメモ機能が使えるようになります。

# 目指した課題解決
## カロリー管理をしたいけどいちいち紙に書いて管理するのは面倒、機能がシンプルな方が使いやすいという方にオススメなカロリーメモアプリです。

# 洗い出した要件

| 機能         | 目的                                   | 詳細                                                  | ストーリー(ユースケース)                                                        |
| ----------- | -------------------------------------- | ---------------------------------------------------- | --------------------------------------------------------------------------- |
| 検索機能     | 食品名でkcalを検索できるようにするため       | 食品名や料理名を入力し検索ボタンを押すとカロリー、gの情報が出る | カロリーを知りたい食品名を検索欄に入力し、検索ボタンを押す                            |
| メモ機能     | 情報を保存できるようにするため              | 検索で得た情報を各入力欄に記入し保存ボタンを押し、保存する      | 検索した食品名、カロリー、gを指定の入力欄に入力し、保存ボタンを押す                    |
| 日付選択機能  | いつ情報を保存したかわかるようにするため      | カランダーから日付を選択しクリックする                      | 日付選択欄をクリックし保存日を選択し保存ボタンを押す                                 |
| 編集機能     | 誤入力した時修正できるようにするため          | ログインしているユーザーが自分の保存した情報を編集できる       |                                                                             |
| 削除機能     | 間違って保存した情報を削除できるようにするため  | ログインしているユーザーが自分の保存した情報を編集できる       |                                                                             |
| ログイン機能  | マイページを作る為                         | ログイン画面に遷移しユーザ登録ができる                      | ログインボタンを押しニックネーム、性別、メールアドレス、パスワードを入力し登録ボタンを押す |
