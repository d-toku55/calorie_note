### アプリケーション名
## Easy Calorie Note

### アプリケーション概要
## このアプリケーションはkcalを知りたい食品名を検索し、検索結果をメモできるアプリです。

### URL
## https://calorie-note.herokuapp.com/

### 利用方法
## kcalを知りたい食品名を検索欄に入力し、検索結果の「食品名、g、kcal」をそれぞれ入力し、保存ボタンをクリックすると保存されます。
